OpenLink Data Spaces

The OpenLink Data Spaces are a powerful Cloud system to manage contacts, calendars, tasks, a social network, emails, bookmarks, news or blog feeds, and much more. ODS supports fine-grained sharing of information through complex access control lists. ODS can interface with virtually any existing personal data managing web service to consolidate data streams from different sources into one coherent interface. Its complete developer API and broad support of standard interfaces make it the perfect personal information management service, either deployed as a shared intranet or internet service or as a personal storage at home. ODS provides all the tools UI developers require to build professional and powerful cloud applications.

ODS is powered by OpenLink's own Virtuso universal server. It runs as a web service directly inside Virtuoso and is easily deployed via Virtuoso's package system.

Try it today, take a feature tour, or check out the ODS API docs.

ODS Highlights

  • Backend
  • Powerful ACL System
  • Standard-compliant
  • Connection
  • Linked Data
  • Open Source

ODS is a powerful Cloud system which is independant from a specific UI and can be deployed everywhere, be it a public server, an intranet installation or a private local desktop system.

ODS features powerful and complex ACL rules which allow to share resources with individuals, fixed groups, or dynamic groups which are defined by an arbitrary set of conditions.

ODS is fully standard-compliant, exposing data through well established interfaces and protocols such as WebDAV, CalDAV, RSS, or RDF.

ODS connects data from third-party services, allows to combine, filter, and expose it again, subject to powerful ACL rules.

ODS exposes all information as linked data which is retrievable either through SPARQL queries or by dereferencing ODS resource URIs.

Get ODS

ODS comes bundled with Virtuoso, OpenLink's universal server. The open-source version of Virtuoso can either be downloaded from the Virtuoso Sourceforge project page or be forked on GitHub.

Alternatively most Linux distributions ship packages for easy Virtuoso open-source installation.
