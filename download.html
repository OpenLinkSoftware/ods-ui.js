Get OpenLink Data Spaces

OpenLink Data Spaces is shipped as part of OpenLink's universal server Virtuoso. It comes as Virtuoso VAD packages which are easily installed.

Get ODS

The open-source version of Virtuoso can either be downloaded from the Virtuoso Sourceforge project page or be forked on GitHub.

Alternatively most Linux distributions ship packages for easy Virtuoso open-source installation.

Installation

ODS is distributed as Virtuoso VAD packages. See the ODS Installation guide in the ODS Wiki for details.

Please note that ODS is preinstalled into the Virtuoso Demo Database. Thus, you do not need to perform the installation steps detailed in the wiki if working with a Virtuoso demonstration database.
